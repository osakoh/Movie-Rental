## Movie Rental Application.

### Requirements
---
User should be able to:
- [x] Create a profile.
- [x] Add new movies to list/load movie(s) from file.
- [x] View list of movies.
- [x] Delete a movie from list.
- [x] Track watched movies.
- [x] View list of watched movies.
